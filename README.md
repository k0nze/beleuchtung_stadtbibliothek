Pin-Belegung `CON1`:

 * 1 => `VCC`
 * 2 => `GND`
 * 3 => `CLK`
 * 4 => Clear für für Schieberegister
 * 5 => Data-Input für Schieberegister 
 * 6 => Enable für LED-Treiber
